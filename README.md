# Penguin-Pen Holder
Hello! 
Actually, initially I actually wanted to design a lamp, but because of 3D printing device constraint and reconsidering the durability of the object I changed it to a pen holder. So, for some context I designed this pen holder to one of my teacher, as later.. around May I will graduate from my school. I don't think that I would be able to see her again..
She likes penguin. I wanted to design a penguin pen holder for her. 
in this project I 3D design a penguin pen holder using Fusion360.

## Key Features
- **Penguin top 3D Design** Equipped with holes for lamp cable, and hole to install the heat set insert.
- **Penguin bottom 3D design**
- **4 Heatset insert** to connect the top and bottom part
